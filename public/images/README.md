# Image Assets

Place your image files here:

- `hero-poster.jpg` - Hero section video poster/fallback (1920x1080)
- `og-image.jpg` - Open Graph social sharing image (1200x630)
- Brand logos, property photos, etc.

## Recommended Specifications

- **Format**: JPG for photos, PNG for graphics with transparency, WebP for optimized delivery
- **Hero images**: 1920x1080 or 2560x1440 for high-DPI displays
- **Thumbnails**: 400x300 minimum
- **OG Image**: 1200x630 for social sharing
